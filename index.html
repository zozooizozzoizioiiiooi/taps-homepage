taps

developed by zozooizozzoizioiiiooi



taps is a mini-game that competes for input speed on the numeric keypad.

When you start the game, you will see a 20-digit number on your screen. Compete for the clear time to enter this number using the numeric keypad. If you make an incorrect input, a penalty of +0.2 second will be added to the clear time. Let's aim for the top of the world ranking!
